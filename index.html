Sistem Alert v1.5

Sistem alert tetap menggunakan konfirmasi Candle Close untuk menghindari repaint.

  • Entry Alert: Bunyi saat candle konfirmasi selesai. Berisi detail harga, TP, SL, dan R:R Ratio.
  • Execution Alert: Notifikasi real-time saat harga menyentuh garis TP atau SL lu.

*Gunakan opsi "Once Per Bar Close" saat membuat alert manual di TradingView.

Cara Setup Cepat

  1. Input Big TF (Hakim Besar) untuk filter trend utama.
  2. Input Small TF (Hakim Kecil) untuk presisi entry.
  3. Sesuaikan ATR Multiplier sesuai volatilitas pair (Gold biasanya butuh multiplier lebih lebar).
  4. Atur Font Size agar tidak menutupi candlestick di layar lu.
